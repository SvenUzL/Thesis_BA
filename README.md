Dieses Verzeichnis enthält die folgenden Verzeichnisse und Dateien:

- Top_35
- Top_5
- bridges_top_n.py


Das Verzeichnis Top_35 beinhaltet die von bridges_top_n.py erzeugte Top‑35‑Auswertung für die wichtigsten Bridges je Monat mit zusätzlichen Sicherheitsansatzklassen pro Protokoll.

Das Verzeichnis Top_5 enthält die von bridges_top_n.py erzeugte Top‑5‑Auswertungen (CSV) für die wichtigsten Bridges je Monat bzw. Jahre.

Die Datei bridges_top_n.py lädt öffentliche Bridge‑Daten von DeFiLlama, erzeugt tägliche Zeitreihen pro Bridge und fasst diese zu Monats‑ und Jahreswerten zusammen. Zusätzlich werden die Top‑N‑Listen (je Monat/Jahr) als separate CSVs exportiert.
